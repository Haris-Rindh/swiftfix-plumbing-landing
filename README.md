SwiftFix Plumbing - High-Converting Landing Page

A fully responsive, high-performance landing page designed for local service businesses (specifically plumbing). This project focuses on lead generation, trust-building, and mobile conversion.

🌟 Key Features

Sticky Emergency Navigation: "Call Now" button always visible for urgent conversions.

Smart Lead Capture: integrated quote form with localStorage logic to remember submission state.

Scroll Animations: Elements fade in and slide up as the user scrolls.

Responsive Design: Optimized for mobile, tablet, and desktop views.

Trust Indicators: Sections for licenses, social proof (reviews), and value propositions.

Performance: Built with semantic HTML and vanilla JS for a 100/100 potential Lighthouse score.

🛠️ Tech Stack

HTML5: Semantic structure.

Tailwind CSS: Utility-first styling (via CDN for easy portability).

Vanilla JavaScript: Lightweight interactivity without heavy frameworks.

🚀 How to Run

Since this is a static site with no build step required:

Clone the repository or download the files.

Open index.html in your web browser.

That's it!

📂 Project Structure

/
├── index.html    # The main landing page file
└── README.md     # Project documentation


🎨 Customization

To adapt this for another business:

Open index.html.

Update the Tailwind Config script tag to change the brand colors (swift.blue, swift.dark).

Replace the <img> URLs with your own business images.
